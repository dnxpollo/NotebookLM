# 📘 Guia de Uso do NotebookLM para Estudos de Economia

Este repositório demonstra como utilizar o **NotebookLM** para organizar estudos sobre **Inflação e Poder de Compra**, aplicando IA como ferramenta de aprendizagem ativa.

---

## ✅ Checklist de Configuração e Uso

- [ ] Criar uma conta no Google( ou usar uma já existente) e acessar o [NotebookLM](https://notebooklm.google/).
- [ ] Abrir um novo caderno temático (`Untitled notebook` → renomear para "Inflação e Poder de Compra"). Renomeie de acordo com o seu assunto. 
- [ ] Adicionar de 3 a 5 fontes abertas (PDFs ou artigos) relacionadas ao tema.
- [ ] Escrever uma **descrição inicial** explicando os objetivos do estudo.
- [ ] Definir **objetivos de estudo** (ex.: entender conceitos básicos de inflação, impactos no poder de compra).
- [ ] Elaborar **perguntas estratégicas** (ex.: "Como a inflação afeta famílias de baixa renda?").
- [ ] Testar diferentes **prompts** (resumo, comparação, glossário, aplicação prática).
- [ ] Registrar as respostas e referências fornecidas pela IA.
- [ ] Organizar os resultados em:
  - Resumos estruturados
  - Glossário de conceitos
  - Conjunto de prompts reutilizáveis
- [ ] Exportar ou documentar os resultados neste repositório.

---

## 💡 Comentários Importantes

- **Não é necessário conhecimento técnico**: o NotebookLM funciona como um chat, basta digitar perguntas ou instruções em texto simples.  
- **Markdown é opcional**: aqui usamos apenas para organizar melhor o guia no GitHub. No NotebookLM, você pode escrever em texto corrido.  
- **Curadoria de fontes**: escolha materiais confiáveis (ex.: Banco Central, artigos acadêmicos, sites de educação financeira).  
- **Pensamento crítico**: não aceite respostas da IA sem reflexão; sempre compare com as fontes originais.
